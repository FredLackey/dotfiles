## Errors

### Error 1: kiro-cli.sh - Unknown option: --no-confirm

- **Phase**: AI Tools installation (kiro-cli.sh)
- **Exit Code**: 1
- **Error Message**:
```
Installing Kiro CLI...
Unknown option: --no-confirm
```
- **Root Cause**: The kiro install script at `https://cli.kiro.dev/install` only accepts `--force`, `--channel`, and `--help`. The `--no-confirm` flag passed in our installer is not a valid option. The script explicitly errors on unknown options via `error "Unknown option: $1"`.
- **Fix Applied**: Removed `-- --no-confirm` from the curl pipe invocation. The kiro Linux installer is non-interactive by default on Linux; no extra flags are needed.
- **Fix Committed**: `c60865b` - `fix(ubuntu-server): remove invalid --no-confirm flag from kiro-cli installer`
